# Mega Adventure

Una aventura épica basada en el Ramayana, donde controlas a Rama para derrotar a las fuerzas del mal.

## Características

- **Combate de Acción**: Lucha contra hordas de Rakshasas.
- **Narrativa**: Sigue la historia épica a través de capítulos.
- **Sistema de Progresión**: Sube de nivel y mejora tus habilidades.
- **Gráficos Estilizados**: Arte visual vibrante y efectos de partículas.

## Tecnologías

- **Vite**: Para un entorno de desarrollo rápido.
- **JavaScript (ES6+)**: Lógica del juego y motor personalizado.
- **Canvas API**: Renderizado de gráficos.

## Instalación y Uso

1.  **Clonar el repositorio:**

    ```bash
    git clone https://github.com/angpa/mega-adventure.git
    cd mega-adventure
    ```

2.  **Instalar dependencias:**

    ```bash
    npm install
    ```

3.  **Iniciar el servidor de desarrollo:**

    ```bash
    npm run dev
    ```

    El juego estará disponible en `http://localhost:5173`.
